git提交出现错误
发表于2022-11-08
在提交github代码时偶然遇见的问题 出现的问题1fatal: unable to access 'https://github.com/heyfriendis/blog.git/': SSL certificate problem: unable to get local issuer certificate 原因及解决办法这是由于当你通过HTTPS访问Git远程仓库的时候，如果服务器上的SSL证书未经过第三方机构认证，git就会报错。原因是因为未知的没有签署过的证书意味着可能存在很大的风险。解决办法就是通过下面的命令将git中的sslverify关掉： 1git config --system http.sslverify false 上面这行命令的影响范围是系统当前用户，如果要设置为全局所有用户，可以改成这样： 1git config --global http.sslverify false 如果只是想针对当前仓库进行设置，可以在需要修改的仓库目录下执行： 1git config http.sslverify false
